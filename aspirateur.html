Description

Le projet Aspirateur Robot repose sur l'utilisation de SLAM (Simultaneous Localization And Mapping) pour permettre une navigation autonome dans un environnement domestique.

Détails Techniques

  • Utilisation de capteurs LIDAR et infrarouges.
  • Intégration avec ROS 2 pour la gestion des algorithmes.
  • Prototypage sur Raspberry Pi.
